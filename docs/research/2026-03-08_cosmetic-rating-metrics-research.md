# Cosmetic Product Rating Metrics Research

> 調査日: 2026-03-08
> 目的: 業界標準の化粧品評価軸を調査し、alche:me のカテゴリ別スペック表示に信頼性の高い用語を採用するため

---

## 1. @cosme (アットコスメ) — 日本最大の美容口コミサイト

### 評価システム
- **7段階おすすめ度** (0~7の8段階)
  - ★7: 最高！自信を持っておすすめ
  - ★6: 最近のヒット！みんなに教えたい
  - ★5: 自分にぴったり。リピート決定
  - ★4: まあまあ。好印象
  - ★3: 普通。良くも悪くもない
  - ★2: 微妙。改善を期待
  - ★1: 合わない。おすすめできない
  - ★0: ダメ。問題あり
- 偶数段階(0含めて8段階)にすることで中心値を排除し、評価の分散を促進する心理学的設計

### 投稿時の必須/任意項目
- **必須**: おすすめ度(★数)、クチコミ本文、購入形態選択
- **任意**: 効果・特徴タグ（カテゴリに応じた選択肢）、画像、使用期間

### 多次元評価の有無
- @cosme は **単一軸 (おすすめ度)** のみをスコア化
- 多次元評価は構造化されていない（フリーテキストで言及）
- 効果・特徴は「タグ」として選択可能（例: 保湿、透明感、カバー力）
- ランキングページでは「お悩み・効果」(約50カテゴリ) でフィルタ可能
  - 毛穴の開き、潤い、ニキビ、くすみ、シワ・たるみ 等

### 特徴
- 総合スコアは独自アルゴリズムで算出（メンバー信頼度重み付け）
- カテゴリ別の構造化された多次元評価は**行っていない**

Sources:
- [クチコミ評価が★7つの理由](https://www.cosme.net/beautist/article/2602931)
- [クチコミ投稿方法](https://www.cosme.net/matome/I0021202/)
- [お悩み・効果ランキング](https://www.cosme.net/ranking/effects)

## 2. LIPS (リップス) — 国内最大級コスメ口コミアプリ

### 評価システム
- **★評価** (具体的な段階数は非公開だが5段階と推定)
- **カテゴリ別特徴タグ評価** (category_features) -- **これが最も参考になるモデル**

### アルゴリズム特徴 (経済学者と共同開発)
1. **購入元バイアス除去**: Mann-Whitney U検定で提供品と購入品の評価歪みを補正
2. **ユーザーレベルバイアス正規化**: 個人の評価傾向を標準化
3. **レビュー数信頼度重み付け**: 少数レビュー商品は信頼度を下げる
4. **アカウント活動度検証**: 低品質投稿・不正アカウントを排除

### カテゴリ別評価タグ (LIPS の核心機能)

LIPS はカテゴリごとに異なる「特徴タグ」を設定し、各商品のクチコミデータから項目別スコアを算出・表示している。

## 3. mybest (マイベスト) — 商品比較検証メディア

プロ監修 + 実機検証で年間1,500点以上のコスメを比較。5段階スコア制。

### ファンデーション検証項目
| 検証項目 | 説明 |
|----------|------|
| カバー力 | 肌診断機器で毛穴・色ムラの隠蔽度を計測 |
| 崩れにくさ | 8時間着用テスト (乾燥崩れ + 皮脂崩れ) |
| 仕上がり | マット/ツヤ、厚塗り感、自然さ |
| テクスチャ/使用感 | 伸び、馴染み、肌触り |
| パッケージ | 携帯性、ミラーサイズ、開閉しやすさ |
| 香り | 不快感の有無 |
| 色展開 | 幅広い肌色対応 |

### 口紅検証項目
| 検証項目 | 説明 |
|----------|------|
| 色持ちのよさ | 飲食後の色残り |
| 乾燥のしにくさ | 唇の保湿維持 |
| 色移りのしにくさ | マスク・グラス等への移り |
| テクスチャの心地よさ | 塗り心地・着用感 |
| においの不快感のなさ | 無香 or 不快でないか |
| カラーバリエーションの豊富さ | 色数 |

### 化粧水検証項目
| 検証項目 | 説明 |
|----------|------|
| 保湿力 | 塗布2時間後の水分増加量 (+30以上で高保湿認定) |
| 成分 | 整肌成分の充実度 |
| 使用感 | しっとり/さっぱりのテクスチャー感 |
| 香り・使いやすさ | 総合的な使用体験 |

Sources:

## 4. 美容雑誌 (VOCE / MAQUIA / 美的)

### ベストコスメアワード
- 年2回 (上半期: 8月号、下半期: 1月号)
- カテゴリ別 (アイシャドウ部門、リップ部門、etc.) で選出
- 価格帯別 (プチプラ / デパコス) 受賞もあり

### 検証方法の特徴
- **VOCE**: 使用感検証 (崩れにくさ検証、目の形別アイシャドウ塗り比較)
- **MAQUIA**: インフルエンサー/有名人のお気に入り紹介中心
- **美的**: ナチュラル/自然派寄りの評価

### LDK the Beauty
- 「忖度なしの本音評価」が売り
- 年間3,000個以上検証、約10,000回テスト
- 機器計測 + 専門家評価の組み合わせ

Sources:
- [美容雑誌のベスコス号読み比べ](https://note.com/miko_2/n/n829ec8a7a1c1)

## 5. 海外プラットフォーム

### Sephora
- **5段階★評価** (単一スコア)
- **is_recommended** (おすすめ/おすすめしない の二値)
- カテゴリ別の構造化多次元評価は**なし**
- レビューフォームではカテゴリに応じた補足質問あり (肌タイプ、肌トーン等)
- 評価はフリーテキスト中心

### Ulta Beauty
- **5段階★評価** (単一スコア)
- 構造化多次元評価は**なし**
- Allure アワードでは: 効果、革新性、テクスチャー、香り、パッケージ、価値 で評価

### Beautylish
- **5段階★評価** (単一スコア)
- コミュニティベースのレビュー、構造化多次元評価は**なし**

### Temptalia (美容レビューサイト)
- 独自の多次元評価 + dupes (類似品比較) 機能

## 6. 化粧品評価の科学的アプローチ

化粧品の評価は大きく2種類に分かれる:

### 機能評価 (Functional Evaluation)
- 客観的・数値化可能: 水分保持量、肌色計測、pH、シワ評価
- 日本: メーカー独自基準 / EU: EEMCO ガイダンス

### 官能評価 (Sensory Evaluation)
- 主観的・人間の知覚: テクスチャーの魅力、使用感の快適さ
- 基準品との相対評価が推奨 (約6項目程度を重点評価)

Sources:
- [化粧品評価の科学](https://rikei-biyouka.com/2019/07/bsseminor/)

## 7. 統合分析: カテゴリ別標準評価軸の提案

### 重要な発見

1. **@cosme と Sephora は単一軸** (おすすめ度 / ★評価のみ) -- 多次元構造化評価はしていない
2. **LIPS が最も洗練されたカテゴリ別多次元評価モデル** を持つ -- alche:me が最も参考にすべきプラットフォーム
3. **mybest は検証メディアとして最も体系的** -- 専門家による検証項目が参考になる
4. **海外プラットフォームは多次元評価に消極的** -- フリーテキストに依存

### LIPS の評価タグ体系から見えるパターン

LIPS はカテゴリを問わず **共通タグのプール** から、カテゴリに適切なタグを選択して表示する方式:

**共通タグ** (全カテゴリまたは大半に出現):
- 保湿、ツヤ、マット、透明感、コストパフォーマンス、持ち運び、大容量、香り、UVカット

**メイクアップ系共通** (リップ/アイシャドウ/ファンデ):
- 発色、キープ力、伸びのよさ、肌なじみ、ラメ

**ベースメイク固有**:
- カバー力、毛穴カバー、崩れにくさ

**スキンケア固有**:
- 肌荒れ予防、毛穴ケア、年齢肌ケア、さっぱり、高機能

### alche:me への推奨: カテゴリ別スペックラベル

## 8. 設計上の注意点

### 薬機法 (旧薬事法) への配慮
- 「浸透力」は薬機法上グレーゾーン -- **「肌なじみ」** が安全な代替用語
- 「美白」は医薬部外品のみ使用可 -- 一般化粧品では **「透明感」** が安全
- 効果を断定する表現 (「シワが消える」等) は不可

### LIPS モデルの適用方針
1. **共通タグプール方式** を採用 (全ラベルの統一管理)
2. カテゴリごとに表示するタグを設定ファイルで制御
3. ユーザー投稿時は任意選択 (必須にしない)
4. 商品詳細画面ではクチコミデータから自動集計して表示

### 最小実装セット (MVP)
カテゴリを跨いで最も汎用性の高い4項目:
1. **保湿** -- ほぼ全カテゴリで有効
2. **キープ力 / 持続力** -- メイク全般
3. **発色** -- カラーコスメ全般
4. **仕上がり** -- ベースメイク全般

## 9. 結論

**LIPS のカテゴリ別特徴タグモデル** が、alche:me が採用すべき最も実績のある業界標準モデルである。理由:

1. 日本市場最大級の口コミデータ (300万件+) に基づく実績
2. 経済学者と共同開発した統計的に堅牢な評価アルゴリズム
3. カテゴリ別に最適化されたタグセット
4. 共通タグプール + カテゴリ選択方式で拡張性が高い
5. @cosme (単一軸) や Sephora (単一軸) より情報量が多い

alche:me では LIPS の用語をそのまま採用することで、日本のコスメユーザーにとって馴染みのある用語体系を提供し、ユーザー信頼を獲得できる。
