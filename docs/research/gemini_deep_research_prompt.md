## プロンプト本文

あなたは美容/コスメティック領域に特化した AI アプリ「alche:me」の UI/UX 改善のための Deep Research を行います。

### alche:me の概要

alche:me は「コスメティック・パラドックス（持っているコスメの70%以上が使われていない問題）」を解決するパーソナル AI ビューティーコンシェルジュアプリです。

**主な機能:**
- AI美容部員とのチャット（メイクアドバイス、レシピ提案）
- コスメ在庫管理（My Cosme — バーコードスキャン/手動登録）
- 欲しいコスメリスト（Next Cosme）
- メイクレシピの保存・閲覧
- Beauty Log（メイク日記）
- 楽天商品検索連携

**技術スタック:** Next.js 15 PWA + Google ADK (Agent Development Kit) + FastAPI + Firebase/Firestore + Cloud Run

**現在のボトムナビゲーション（5タブ）:**
1. ✨ AI美容部員（チャット）
2. 📦 My Cosme（在庫一覧）
3. 📖 Recipe（レシピ一覧）
4. 🛍️ Next Cosme（欲しいものリスト）
5. 💕 日記（Beauty Log）

**ターゲットユーザー:** 20〜30代女性、日本市場

### 既に Claude Code で実施した調査の概要

以下の11アプリ/サービスを5カテゴリで調査済みです。**Gemini には、この調査結果の検証・補完・最新情報の追加をお願いします。**

#### 調査済みの3つの主要インサイト

1. **ナビゲーション再構築が最優先** — TikTok/Instagram/LIPS/@cosme 全てが「マイページ」を右端タブに配置。中央に「+」ボタンを配置するパターンが業界標準
2. **Beauty Log は「写真ファースト + 自動補完」で習慣化** — ほぼ日手帳の「書かなくてもOK」思想 + Day One の自動メタデータ + Instagram Stories の投稿ハードル低減
3. **ドメイン特化リッチカードが最大の差別化** — Gemini の Flights/Hotels カードのようなドメイン特化が差別化の鍵

#### 提案済みのタブ構成

| タブ | ラベル | 内容 |
|------|--------|------|
| 1 | AI美容部員 | チャット + 上部ストーリーズトレイ |
| 2 | Recipe | Explore 風モザイクグリッド + カテゴリフィルター |
| 3（中央） | **+** | アクションシート: スキャン / 日記 / 手動登録 / 楽天検索 |
| 4 | My Cosme | 在庫 + Next Cosme（セグメントタブ: 持ってる / 欲しい） |
| 5 | マイページ | プロフィール + 設定 + Beauty Log カレンダー + 統計 |

上記5テーマの調査結果を以下の構成でまとめてください:

1. **エグゼクティブサマリー** — 5テーマの主要発見を各1-2行で要約
