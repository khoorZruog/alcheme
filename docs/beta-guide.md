# alche:me β テストガイド

## アクセスURL

| サービス | URL |
|---------|-----|
| Web (フロントエンド) | `https://alcheme-web-<hash>-an.a.run.app` |
| Agent (内部API) | `https://alcheme-agent-<hash>-an.a.run.app` (非公開) |

> **注意:** Phase 1 では Cloud Run のデフォルト URL（*.run.app）を使用します。カスタムドメインは Phase 2 で設定予定です。

---

## テストアカウント作成手順

1. 上記の Web URL にアクセス
2. 「Googleでログイン」ボタンをタップ
3. Google アカウントでログイン
4. オンボーディング画面でプロフィールを設定:
   - ニックネーム
   - 肌タイプ・肌トーン
   - パーソナルカラー
   - エージェントテーマ（親友 / K-POP / メイド）

---

## テストシナリオ

### 1. コスメスキャン
1. ホーム画面の「コスメを鑑定する」をタップ
2. コスメの写真を撮影またはアップロード
3. AI が鑑定結果を表示（ブランド・商品名・レアリティ）
4. 「在庫に登録」で保存

### 2. 在庫一覧
1. ボトムナビの「在庫」をタップ
2. カテゴリフィルタで絞り込み（Lip / Eye / Cheek / Base）
3. カードをタップして詳細表示

### 3. メイクレシピ生成
1. ボトムナビの「チャット」をタップ
2. 「今日のオフィスメイクを提案して」などメッセージ送信
3. AI がストリーミングで回答
4. レシピカードが表示されたらタップで詳細

### 4. レシピフィードバック
1. レシピ詳細画面で「良い / 微妙 / 合わない」を選択

## Phase 1 制限事項

- **ユーザー数:** 最大 50 名（β テスト枠）
- **セッション:** インメモリ管理（サーバー再起動でチャット履歴リセット）
- **楽天API:** 商品検索は日本国内のみ対応
- **画像認識:** 1 回のスキャンで最大 5 アイテムまで
- **レシピ生成:** 手持ちコスメのみ使用（推奨購入は Phase 2）
- **PWA:** オフライン時は静的ページのみ閲覧可能
- **カスタムドメイン:** 未設定（*.run.app URL を使用）
