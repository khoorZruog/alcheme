## テストアカウント作成手順

1. 上記の Web URL にアクセス
2. 「Googleでログイン」ボタンをタップ（またはメール/パスワードで新規登録）
3. Google アカウントでログイン
4. オンボーディング画面でプロフィールを設定:
   - ニックネーム
   - 肌タイプ・肌トーン・パーソナルカラー（スキップ可能）

## 主要テストシナリオ

### 1. コスメ登録（4つの方法）
1. BottomNav の「コスメ」タブ → 右上「＋」ボタンをタップ
2. 登録方法を選択:
   - **スキャン鑑定**: コスメ写真を最大4枚アップロード → AI が自動鑑定
   - **楽天で検索**: 商品名で楽天API検索 → 結果から選択して登録
   - **Web検索**: Agent Server 経由でWeb検索 → 結果から選択して登録
   - **手動で登録**: ブランド・商品名・カテゴリ等を手入力
3. 鑑定結果を確認・修正して「登録する」

### 2. 在庫管理
1. BottomNav の「コスメ」タブで在庫一覧を表示
2. カテゴリフィルタ、ブランドフィルタ、テクスチャフィルタで絞り込み
3. グリッド/リスト切替、5種のソートで並び替え
4. カードをタップして詳細表示
5. 一括選択 → 一括削除・一括残量更新

### 3. AIチャット・メイクレシピ生成
1. BottomNav の「チャット」（✨）タブをタップ
2. 「今日のオフィスメイクを提案して」などメッセージ送信
3. ThinkingIndicator でエージェントの進捗をリアルタイム確認
4. テーマ提案（3択カード）→ タップでレシピ生成
5. レシピカード（MATCHスコア + ステップ一覧）がインライン表示
6. プレビュー画像（AI生成イラスト）が表示される
7. 「詳しく見る」でレシピ詳細ページに遷移

### 4. ショッピング相談
1. 「KATE リップモンスターの05番を買おうか迷ってます」→ 相性レポート
2. 「CEZANNEのチークとCANMAKEのチーク、どっちがいい？」→ 比較テーブル

### 5. レシピ管理
1. BottomNav の「レシピ」タブで一覧表示
2. レシピ詳細: ステップ一覧、プレビュー画像、プロのコツ
3. お気に入り登録、フィードに公開/非公開、削除
4. フィードバック（良い / 微妙 / 合わない）

### 6. Next Cosme
1. BottomNav の「Next Cosme」（🛍️）タブで候補一覧
2. AI提案の Next Cosme アイテム（推薦回数つき）
3. ステータス変更（候補 → 購入済み / 見送り）
4. 手動追加、2-3件の比較機能

### 7. メイク日記（Beauty Log）
1. BottomNav の「日記」（💕）タブでカレンダー表示
2. 日付をタップしてログ作成（ムード、天気、写真、使用レシピ、評価）
3. タイムライン表示で過去のログを振り返り

### 8. ソーシャルフィード
1. サイドメニューまたはプロフィールからフィードにアクセス
2. 公開レシピ投稿の閲覧、いいね、コメント
3. 他ユーザーをフォロー →「Following」タブで絞り込み

### 9. プロフィール・設定
1. 設定画面: 基本情報、美容プロフィール、興味関心、エージェントテーマ、SNSリンクの5項目を編集
2. プロフィール写真の変更（Cloud Storage アップロード）
3. プロフィール完了度バッジの確認

## 既知の制限事項・バグ

### 制限事項
- **ユーザー数:** 最大 50 名（β テスト枠）
- **楽天API:** 商品検索は日本国内のみ対応
- **画像認識:** 1 回のスキャンで最大 4 アイテムまで
- **PWA:** オフライン時は静的ページのみ閲覧可能
- **カスタムドメイン:** 未設定（*.run.app URL を使用）
- **Memory:** AIの学習はセッション内のみ（コンテナ再起動でリセット）

### 既知のバグ
- **天気API**: 天気情報の取得に失敗する場合がある（「天気情報の取得に失敗しました」）
- ~~**Next Cosme 手動追加**: 手動で Next Cosme を追加しようとすると「追加に失敗しました」になる~~ → **解決済み（2026-02-17）**
