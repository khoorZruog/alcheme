## 機能概要

チャット画面で写真を添付すると、AI美容部員が**画像の中身を理解**して応答する。Google Gemini のマルチモーダル Vision API を活用し、コスメのパッケージ文字・色・形状・テクスチャを直接分析する。

---

## ユースケース

### 1. コスメ鑑定（スキャン登録）

| 項目 | 詳細 |
|------|------|
| **シーン** | 手持ちのコスメを撮影して「これ何？」と聞く |
| **デフォルトプロンプト** | テキストなしで画像送信 →「この画像のコスメを鑑定してください」が自動付与 |
| **AI の動作** | Concierge → Inventory Agent に委譲。パッケージのテキスト・ロゴ・形状・色から**ブランド名・商品名・カテゴリ・色番号・テクスチャ・残量**を推定 |
| **訴求ポイント** | 写真を撮るだけで在庫登録完了。手入力不要 |
| **マーケコピー例** | 「写真1枚でコスメを自動認識。AI美容部員があなたのポーチを瞬時にデータ化」 |

### 2. メイクルック再現

| 項目 | 詳細 |
|------|------|
| **シーン** | SNSで見たメイク写真や雑誌の切り抜きを送って「このメイクを再現したい」 |
| **AI の動作** | Gemini Vision が色味・質感・パーツ配分を分析 → 手持ちコスメで再現するレシピを提案 |
| **訴求ポイント** | 「見た目」から「手順」に変換。憧れのメイクが手持ちコスメで実現 |
| **マーケコピー例** | 「気になるメイクの写真を送るだけ。手持ちコスメで再現レシピをAIが即提案」 |

### 3. 今日のメイク記録（ビフォーアフター）

| 項目 | 詳細 |
|------|------|
| **シーン** | 朝のメイク完成後の自撮りを送って「今日のメイク、記録して」 |
| **AI の動作** | 画像をメイク日記（Beauty Log）と紐づけて保存。評価やアドバイスも付与 |
| **訴求ポイント** | 写真ベースの日記で振り返りが楽しい |
| **マーケコピー例** | 「毎朝の1枚が、あなただけのビューティジャーナルに」 |

### 4. コスメ比較・相談

| 項目 | 詳細 |
|------|------|
| **シーン** | 店頭で2つのリップを並べて撮影「どっちが似合う？」 |
| **AI の動作** | 画像内の色味を分析 + ユーザーのパーソナルカラー情報と照合 → 似合う方を提案 |
| **訴求ポイント** | 店頭での迷いをAIが即解決。一人でも安心して買い物 |
| **マーケコピー例** | 「迷ったら撮って聞くだけ。パーソナルカラーに合う1本をAIが即答」 |

## 技術的な裏付け（マーケ資料用）

- **AI モデル**: Google Gemini 2.5 Flash（マルチモーダル対応）
- **画像処理**: Base64エンコード → Gemini Vision API にダイレクト送信
- **分析精度**: パッケージ文字読み取り → デザイン/形状推定 → 色彩分析の3段階
- **レスポンス**: リアルタイムSSEストリーミング（打ち返しアニメーション付き）
